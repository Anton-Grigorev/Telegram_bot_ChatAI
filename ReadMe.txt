Телеграм бот ChatAI
Создан для удобства обращения к ChatAI (ИИ) и профессионального совершенствования в написания исходного кода.
За основу взята модель: "text-davinci-003" от AI. Настройку количества символов в ответе увеличил до 1000.
Необходимо импортировать следующие библиотеки и токены, API:
- openai
- aiogram
- Токен BotFather
- API от AI
Запустить можно в любой среде разработки.
